# Godot---Duck-Hunt

Projeto de recriação do clássico jogo Duck Hunt (Atari/Nintendo) feito em **Godot Engine**, mesclando elementos das versões 3.0 e 4.0.

## Sobre o projeto
Clone simples para praticar mecânicas 2D: mira usando o mouse, tiro, patos voando aleatoriamente, score e game over.

## Gameplay
Dunk Hunt rodando (https://imgur.com/a/07rv2Ko)

Baseado no curso de Godot do Danki Code (créditos: https://cursos.dankicode.com/curso-godot-completo, https://www.youtube.com/@DankiCode).
**O que fiz nesse projeto**:
- Segui as aulas iniciais para construir a base do jogo clássico Duck Hunt.
- Atualizei praticamente todo o código da versão Godot 3.x para Godot 4.x (signals novos, physics atualizados, typed arrays, export vars, etc.).
- Corrigi bugs de compatibilidade e melhorei organização de cenas/scripts com ajuda de pesquisas e ferramentas de IA (Grok).

## O que aprendi
- Física 2D e movimento de sprites.
- Input de mouse 
- Colisões e detecção de acertos(audios).
- UI simples (score(não há menu).
- Mescla de Godot 3.x e 4.x (ex: signals, nodes atualizados).
- Como lidar com breaking changes (ex: signals, nodes, input system).
- Boas práticas de organização de cenas e scripts.
- Uso de IA como apoio para resolver problemas técnicos.
